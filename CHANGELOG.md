# Changelog

## [0.1.0] - 2025-06-02 ### Initial
- Support functionality: Client interface, Order, Payment, Payout (single and batch), WebhookVerifier.
- Support for `platform_fees` and `payee` in Orders API v2 to accommodate marketplace scenarios.
- Basic configurations.
- Basic error handling.
